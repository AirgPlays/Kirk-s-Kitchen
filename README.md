# Kiri's Kitchen Website

This is a mini-project from Code.org's **Lesson 5: HTML Web Page** (Intro to Computer Science). The goal of this project was to create a customer website based on a specific client request.

The project was completed on **October 22, 2024**.

---

## Project Overview

The website represents **Kiri's Kitchen**, a cozy family-owned restaurant in San Francisco. It includes:

- A **banner** with special promotions
- **Main content** featuring:
  - Restaurant introduction
  - Menu highlights (Appetizers, Main Courses, Desserts, Drinks, Seasonal Items)
  - Customer testimonials
  - Images of the restaurant
- **Footer** with contact information and hours

## Technologies Used

- **HTML5** – Structure of the web page
- **CSS** – Styling the layout, banner, and sections
- External stylesheet linked via `style.css`

## Features

- Fixed banner at the top with promotion information
- Organized menu section with categories
- Testimonials section with images
- Contact info footer
- Semantic HTML structure using `<header>`, `<main>`, `<section>`, and `<footer>`
